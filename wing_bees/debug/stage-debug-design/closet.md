Debug analyzes pipeline failures and proposes fixes. Fixes are categorized: inline (single-line obvious), backlog items, or escalate-to-plan (for larger problems). Escalate-to-plan hands off to a fresh plan stage with the failure as briefing. Per-repo inline-fix authority is governed by the queen-file; default requires operator approval before applying.
