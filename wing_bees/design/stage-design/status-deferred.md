## Status: DEFERRED

This stage/feature is not yet implemented in bees. Content is preserved here for future reference.

A freestanding, occasional activity — not a pipeline stage. Run it when you need to think through a big feature, a refactor, or an open architectural question before (or instead of) committing to a plan. Design sessions are interactive and may pull in issues, backlog items, or existing design docs as raw material. Output feeds into plan briefings when implementation should follow, but design does not have to precede plan and a plan does not require a prior design run.
