`gh pr create --title <title> --body <body> --base <base> [--draft]` opens a PR from the current branch. `gh pr view` reads PR state (useful for status checks). `gh issue create --title <title> --body <body> [--label <label>]` files issues. Link issues with `Closes #N`, `Fixes #N`, or `Refs #N` in PR or commit bodies — GitHub auto-links them and closes issues on merge.
