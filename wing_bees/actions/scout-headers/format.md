## Format

The header format the scout writes is the generic `agentic-file-headers` pattern: path+role line, 2–5 sentence description, key exports for code files, ISO-8601 UTC timestamp. See `wing_practices/agentic-file-headers` for the language-specific comment syntax table, full examples, and field definitions. Scout's only bees-specific contribution is the field name (`Scout: …`) and the staleness-detection mechanism above.
