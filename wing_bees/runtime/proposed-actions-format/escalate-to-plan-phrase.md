## Escalate-to-plan phrase

The canonical phrase for debug → plan handoff is **"Escalate to plan."** Builders and retro/debug queens use this phrase when a problem exceeds their scope. The operator sees it in proposed-actions and understands it means a new plan-spec-dispatch cycle.

If a debug escalation itself fails, there is a loop guard so the fix-of-the-fix doesn't trigger recursive escalation.
