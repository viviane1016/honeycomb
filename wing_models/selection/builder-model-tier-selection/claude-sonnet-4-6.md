## claude-sonnet-4-6

### Closet

Default builder tier; reliable for typical Python edits, test additions, and well-scoped refactors; use when the spec is clear and the scope is bounded.

### Drawer

Observed reliability across the bees codebase is strong. Sonnet handles typical Python edits, test additions, and well-scoped refactors with consistent results. The local sonnet-split A/B (BEES_SONNET_LOCAL_SPLIT) ran in v0.1 and is now removed, simplifying the builder tier selection.
